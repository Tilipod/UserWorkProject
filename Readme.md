Проект "UserWorker".

Позволяет выполнить 5 запросов на Spring RESTful-сервер (Java 8, база данных PostgreSQL):
1) Добавить картинку на сервер (POST /sources/images). Возврат URI сохраненной картинки
2) Добавить нового пользователя на сервер (POST /users). Возврат ID созданного пользователя
3) Получить информацию о пользователе по ID (GET /users/{id})
4) Изменить статус пользователя online/offline (POST /users/{id}). Возврат ID пользователя, старого и нового статусов
5) Получить статистику сервера (пользователи с указанным статусом, который в последний раз изменяли после указанной метки). 
   Возврат всех подходящих пользователей. (GET /users). Допустимые параметры фильтрации: isOnline и timestamp
   
Все параметры (кроме ID) клиент передает в формате JSON. 
Запросы 3 и 5 приоритезированы.
Все запросы исполняются в отдельных потоках.
Используется логирование ошибок.
